## Get Merchant Transactions
Получение списка транзакций конкретного мерчанта.


## Path Parameters

| Parameter | Type | Required | Description |
|----------|-------|----------|-------------|
| ```id```  | string  | Yes  | Unique identifier of the merchant |

## Request Body

| Parameter | Type | Description |
|----------|-------|-------------|
| ```amount``` | number  | Transaction amount |
| ```card``` | integer  | Card identifier |
| ```changed_by``` | integer  | ID of user who last modified the transaction |
| ```changed_on``` | string  | Timestamp of last modification (ISO 8601) |
| ```created_by``` | integer  | ID of user who created the transaction |
| ```created_on``` | string | Creation timestamp (ISO 8601) |
| ```currency``` | integer | Currency identifier |
| ```currency_rate``` | number | Exchange rate applied to the transaction |
| ```merchant```  | integer | Merchant identifier |
| ```merchant_order_id``` | string | Merchant's internal order ID |
| ```merchant_payload``` | string  | Additional merchant data (nullable) |
| ```order_group_id```  | string | Group identifier for related orders |
| ```order_id```  | string | Unique order identifier |
| ```reciept_link```  | string | URL to transaction receipt |
| ```status```  | integer | Transaction status code |
| ```team```  | integer | Unique identifier of the merchant |
| ```type```  | integer | Transaction type code |
| ```username``` | string | Username associated with transaction |
