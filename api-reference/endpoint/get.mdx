## Get Merchant Detais
Retrieves detailed information about a specific merchant by their ID.


## Path Parameters
| Parameter | Type | Required | Description |
|----------|-------|----------|-------------|
| ```id```  | string  | Yes  | Unique identifier of the merchant |

## Response
```json
{
    "address": string,
    "callback_url": string,
    "cents_allowed": boolean,
    "changed_on": string (ISO 8601 datetime),
    "created_on": string (ISO 8601 datetime),
    "display_name": string,
    "fs_uniquifier": string,
    "name": string,
    "registration_date": string (YYYY-MM-DD),
    "return_urls": string (JSON),
    "secret_key": string
}
```

## Response Fields

| Field | Type | Description |
|----------|-------|----------|-------------|
| ```address```  | string | Physical or registered address of the merchant |
| ```callback_url``` | string | URL that receives transaction notifications |
| ```cents_allowed``` | boolean | Indicates if cents/decimal amounts are allowed in transactions |
| ```changed_on``` | string | Timestamp of the last modification (ISO 8601 format) |
| ```created_on``` | string | Timestamp when the merchant was created (ISO 8601 format) |
| ```display_name``` | string | Human-readable name of the merchant |
| ```fs_uniquifier``` | string | Unique identifier for file system operations |
| ```name``` | string | Yes | Machine-readable merchant identifier |
| ```registration_date``` | string | Date when the merchant was registered (YYYY-MM-DD format) |
| ```return_urls``` | string | JSON string containing return URLs configuration |
| ```secret_key``` | string  | Secret key used for API authentication |
