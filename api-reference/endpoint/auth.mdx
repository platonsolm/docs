## Login
Аутентифицируйтесь, чтобы получить токен доступа, который можно использовать для последующих запросов.

## Endpoint
```
POST /api/v1/security/login
```

## Request Body

| Parameter | Type | Required | Description |
|----------|-------|----------|-------------|
| ```username```  | string  | yes  | Your username |
| ```password``` | string  | yes  | Your password |
| ```provider``` | string | yes  | Authentication provider (use "db") |

<ParamField body="username" type="string">
    username
</ParamField>

<ParamField body="password" type="string">
    password
</ParamField>

<ParamField body="provider" type="string">
    provider
</ParamField>
## Response

<CodeGroup>
```json Response
{
    "access_token": "ey......"
}
```
| Parameter | Type | Description |
|----------|-------|-------------|
| ```access_token``` | string | JWT token to be used for authentication |

</CodeGroup>


## Request
<CodeGroup>
```cURL cURL
curl -X POST https://api.example.com/api/v1/security/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin",
    "password": "testPassword",
    "provider": "db"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.example.com/api/v1/security/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    username: 'admin',
    password: 'testPassword',
    provider: 'db'
  }),
});

const data = await response.json();
// Use the token in subsequent requests
const token = data.access_token;
```

```python Python
import requests

response = requests.post(
    'https://api.example.com/api/v1/security/login',
    json={
        'username': 'admin',
        'password': 'testPassword',
        'provider': 'db'
    }
)

token = response.json()['access_token']
```
</CodeGroup>
