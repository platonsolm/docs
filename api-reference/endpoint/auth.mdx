## Login
Аутентифицируйтесь, чтобы получить токен доступа, который можно использовать для последующих запросов.

## Endpoint
```
POST /api/v1/security/login
```

## Request Body

| Parameter | Type | Required | Description |
|----------|-------|----------|-------------|
| ```username```  | string  | yes  | Your username |
| ```password``` | string  | yes  | Your password |
| ```provider``` | string | yes  | Authentication provider (use "db") |

## Response

<CodeGroup>
```json Response
{
    "access_token": "ey......"
}
```
| Parameter | Type | Description |
|----------|-------|-------------|
| ```access_token``` | string | JWT token to be used for authentication |

## Response
<CodeGroup>
```bash Example Request
curl -X POST https://api.example.com/api/v1/security/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin",
    "password": "testPassword",
    "provider": "db"
  }'
```

```javascript example.js
const response = await fetch('https://api.example.com/api/v1/security/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    username: 'admin',
    password: 'testPassword',
    provider: 'db'
  }),
});

const data = await response.json();
// Use the token in subsequent requests
const token = data.access_token;
```
